# falafel

Flutter e-Commerce application for a restaurant using Provider, http, and shared preferences.

Splash screen with smooth animation. Login and register screens with form validation and shared preferences to save userâ€™s token to see if he/she logged before so the app navigates directly to the home screen. Beautiful home screen with search bar, slider, categories, and products. The bottom navigation bar moves fast between screens. 3 screens one for orders, one for favorites, and one for offers. And finally, the settings screen so you can change your data or sign out.
